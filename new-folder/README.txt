Glance Limousine split front-end files

Files:
- index.html
- styles.css
- app.js

How to use:
1. Put all 3 files in the same folder.
2. Open index.html in a browser for preview.
3. Upload all 3 files to your server together.
4. Replace placeholder/demo integrations with your live backend, payment, and form handling.

Notes:
- This is the split version of the single-file site.
- External fonts and Font Awesome are loaded from CDN.
- Payment buttons are demo front-end actions and should be connected by your developer.
